# BizYair Frontend

# Getting Started

``` bash
git clone https://github.com/siliconflow/bizyair_frontend.git
cd bizyair_frontend 
# 安装依赖
npm install
# prettier 
npm run format 
# eslint 
npx eslint src
# release frontend 
npm run dev
#重新打包前端得到.whl包
npm run build:py
#不重新生成forntend_bizyair.js只重新打包bizyui目录得到.whl包
npm run build:bizyui
```
